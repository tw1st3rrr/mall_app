# Промпт — Изменения сайта афиши + Админка

Внеси следующие изменения в проект сайта афиши мероприятий.

---

## 1. ФИЛЬТРАЦИЯ В АФИШЕ — ВЫПАДАЮЩИЙ СПИСОК ТЦ С ЧЕКБОКСАМИ

В разделе фильтрации мероприятий заменить текущий фильтр по ТЦ на выпадающий список с чекбоксами:

- Кнопка-триггер: `Торговый центр ▾` — при клике открывается дропдаун
- Внутри дропдауна: список всех ТЦ с чекбоксами, кнопки `Выбрать все` / `Сбросить`
- Можно выбрать несколько ТЦ одновременно
- На кнопке-триггере показывать счётчик выбранных: `Торговый центр (3) ▾`
- При выборе — афиша фильтруется в реальном времени без перезагрузки страницы
- Закрытие дропдауна: клик вне области или повторный клик на триггер

---

## 2. ФОРМА ЗАЯВКИ НА АРЕНДУ — ПЕРЕНЕСТИ ВЫБОР ТЦ В ФОРМУ

- Убрать выбор ТЦ из текущего места (шапка / отдельный блок)
- Добавить поле выбора ТЦ прямо внутрь формы заявки на аренду помещения
- Тип поля: select или radio-кнопки (в зависимости от количества ТЦ)
- Поле обязательное, с валидацией перед отправкой
- Расположить поле логично — после контактных данных, перед описанием мероприятия

## 3. ПЕРЕИМЕНОВАНИЕ

- Везде в интерфейсе, заголовках страниц, метатегах и хлебных крошках:
  - `Enka Project Hall` → `Кунцево Плаза`

- Проверить все JSON/данные с названиями и обновить там тоже

## 4. ФОТОГРАФИИ ДЛЯ КАРТОЧЕК МЕРОПРИЯТИЙ

Для каждого типа мероприятия — своя дефолтная обложка. Найти подходящие фото через Unsplash API (бесплатно, без ключа для базовых запросов) и заполнить все пропуски.

### Типы мероприятий и поисковые запросы для Unsplash:

```javascript
const EVENT_TYPE_PHOTOS = {
  "Концерт":           "concert stage lights crowd",
  "Детский праздник":  "children birthday party balloons",
  "Мастер-класс":      "workshop craft hands creative",
  "Выставка":          "art exhibition gallery modern",
  "Спектакль":         "theater performance stage curtain",
  "Кино":              "cinema movie popcorn screen",
  "Фестиваль":         "festival outdoor crowd colorful",
  "Спортивное":        "sports competition arena crowd",
  "Корпоратив":        "corporate event business hall",
  "Лекция / Форум":    "conference speaker audience hall",
  "Ярмарка":           "market fair stalls shopping",
  "Розыгрыш / Акция":  "prize winner celebration confetti",
  "Для всей семьи":    "family fun park children adults",
  "Танцы":             "dance performance stage spotlight",
  "Фотовыставка":      "photo exhibition prints gallery",
};

// Unsplash Source API (без авторизации):
// https://source.unsplash.com/800x450/?{query}
```

### Логика подстановки фото:
1. Если у мероприятия есть своя загруженная фотография → использовать её
2. Если фото нет → определить тип мероприятия → подставить дефолтное фото типа
3. Если тип не определён → использовать общий fallback: `"event hall venue"`

### Заполнить все пропуски:
- Пройтись по всем мероприятиям в данных
- Найти те, у которых нет фото или фото битое
- Подставить дефолтное по типу
- Показать в карточке с лёгким затемнением если фото дефолтное (чтобы отличить от загруженного)

## 5. ТИП МЕРОПРИЯТИЯ НА КАРТОЧКЕ И СТРАНИЦЕ

**На карточке в афише:**
- Добавить бейдж с типом мероприятия в верхний левый угол карточки
- Стиль: цветной тег, цвет зависит от типа (каждому типу свой цвет)
- Пример: `🎵 Концерт` / `🎨 Мастер-класс` / `🎪 Детский праздник`

**На странице мероприятия:**
- Добавить тип мероприятия под заголовком
- Кликабельный — при клике переходит в афишу с фильтром по этому типу

## 6. АДМИН-ПАНЕЛЬ ДЛЯ УПРАВЛЕНИЯ КОНТЕНТОМ

Создать отдельную страницу `/admin` с паролем для входа (хардкод пароля в конфиге на первом этапе).

### 6.1 Управление мероприятиями

**Список мероприятий:**
- Таблица со всеми мероприятиями: название, ТЦ, тип, дата, статус (активно/архив)
- Поиск по названию
- Фильтр по ТЦ и типу
- Кнопки: `Редактировать` / `Удалить` / `Дублировать`

**Форма добавления/редактирования мероприятия:**
- Название мероприятия
- Тип мероприятия (select из предопределённых типов)
- ТЦ (select)
- Дата и время начала / окончания
- Описание (textarea)
- Загрузка фото (drag & drop + preview)
- Ссылка на покупку билетов / регистрацию
- Статус: Активно / Архив / Черновик
- Кнопки: `Сохранить` / `Отмена`

### 6.2 Управление фотографиями

- Галерея всех загруженных фото в проекте
- Возможность заменить фото для любого мероприятия
- Просмотр какие мероприятия используют дефолтные фото (подсвечивать)
- Кнопка `Заменить дефолтное` для быстрой загрузки своего фото

### 6.3 Управление заявками на аренду

- Таблица всех входящих заявок: дата, имя, ТЦ, тип мероприятия, статус
- Статусы: `Новая` / `В обработке` / `Подтверждена` / `Отклонена`
- Возможность менять статус заявки
- Просмотр полного текста заявки
- Экспорт заявок в CSV

### 6.4 Технические требования админки:
- Защита роутом `/admin` с проверкой пароля (localStorage token)
- Все изменения сохраняются в JSON файлы (или localStorage если нет бэкенда)
- Адаптивный интерфейс
- Подтверждение перед удалением (`Вы уверены?`)
