# Промпт для Claude Code: два точечных изменения в проекте

---

## Задача 1 — Убрать навигационный бар с лишними разделами на сайте афиши

### Что сделать

Найди в проекте навигационный бар (header / nav) сайта афиши. В нём сейчас есть разделы которые не относятся к афише мероприятий — например: «Магазины», «Кафе и рестораны», «Развлечения» (и возможно другие похожие).

Удали эти разделы из навигации. Оставь только то что относится непосредственно к мероприятиям и детскому клубу.

### Как найти

1. Открой все HTML-файлы сайта афиши (не лендингов)
2. Найди `<nav>` или `<header>` с навигационными ссылками
3. Найди также все JS-файлы которые могут динамически рендерить навигацию
4. Удали только лишние пункты меню — не трогай структуру, стили и остальной код

### Проверь что после удаления

- Навигация визуально не сломалась (нет пустых мест, отступы корректны)
- Оставшиеся пункты по-прежнему работают
- На мобильном адаптиве навигация тоже корректна

## Задача 2 — Заполнить пустое место справа в форме бронирования билета

### Контекст

На странице отдельного мероприятия внизу есть форма бронирования билета. Она сейчас выглядит так:
- Слева — форма заявки с полями
- Справа — **пустое место**

Нужно заполнить правую часть блоком с картой и контактами — точно по образцу на скриншоте.

### Что добавить справа (сверху вниз)

#### 1. Встроенная карта Яндекс
- Embed через `<iframe>` Яндекс Карт
- Карта показывает местоположение конкретного ТЦ (в зависимости от того, к какому ТЦ относится мероприятие)
- Размер: занимает верхние ~55% правой колонки
- Скруглённые углы сверху (как у карточки)
- Без лишних контролов — только карта с меткой и кнопкой «Открыть в Яндекс Картах»

Iframe-код для каждого ТЦ (подставить нужный в зависимости от мероприятия):

```html
<!-- Кунцево Плаза -->
<iframe
  src="https://yandex.ru/map-widget/v1/?ll=37.361&z=16&pt=37.361,55.730"
  width="100%" height="100%"
  frameborder="0" allowfullscreen
  style="border-radius: 16px 16px 0 0;">
</iframe>

<!-- Каширская Плаза -->
<iframe
  src="https://yandex.ru/map-widget/v1/?ll=37.659&z=16&pt=37.659,55.637"
  width="100%" height="100%"
  frameborder="0" allowfullscreen
  style="border-radius: 16px 16px 0 0;">
</iframe>

<!-- Капитолий -->
<iframe
  src="https://yandex.ru/map-widget/v1/?ll=37.536&z=16&pt=37.536,55.747"
  width="100%" height="100%"
  frameborder="0" allowfullscreen
  style="border-radius: 16px 16px 0 0;">
</iframe>
```

⚠️ Уточни точные координаты каждого ТЦ по реальным адресам из проекта и скорректируй параметры `ll` и `pt` в iframe.

#### 2. Блок с контактной информацией
Расположен под картой, в нижних ~45% правой колонки. Белый или кремовый фон, скруглённые углы снизу.

Содержимое (три строки):

```
📍  [Адрес ТЦ] (например: Москва, Рябиновая ул., 26, ТЦ Кунцево Плаза, 3 этаж)

📞  [Телефон ТЦ] — крупно, выделен цветом акцента (фиолетовый / цвет темы)

🕐  Режим работы: пн–вс 10:00–22:00
```

Данные подставляются динамически в зависимости от ТЦ мероприятия — вынеси их в конфиг:

```js
const VENUE_CONTACTS = {
  kuntsevo: {
    address: "Москва, Рябиновая ул., 26, ТЦ Кунцево Плаза, 3 этаж",
    phone: "+7 (495) 000-00-00",
    hours: "пн–вс 10:00–22:00",
    mapSrc: "https://yandex.ru/map-widget/v1/?ll=37.361&z=16&pt=37.361,55.730"
  },
  kashirskaya: {
    address: "...",
    phone: "...",
    hours: "пн–вс 10:00–22:00",
    mapSrc: "..."
  },
  kapitoliy: {
    address: "...",
    phone: "...",
    hours: "пн–вс 10:00–22:00",
    mapSrc: "..."
  }
};
```

### Стилизация правой колонки

Точно по скриншоту:
- Общий контейнер: белый фон, скруглённые углы (~16px), тонкая граница (1px solid #e0e0e0), тень как у левой карточки с формой
- Карта и контакты — внутри одного контейнера, разделены тонкой линией
- Иконки (📍, 📞, 🕐) — можно SVG или unicode, цвет акцента для телефона
- Телефон — крупный шрифт (~20px), жирный, цвет темы ТЦ
- Адрес и часы — обычный размер (~14px), серый цвет

### Вёрстка двух колонок

Проверь что layout двух колонок (форма слева, карта+контакты справа) корректно адаптируется:
- **Десктоп**: две колонки рядом, одинаковой высоты (`align-items: stretch`)
- **Мобильный** (< 768px): колонки складываются в одну — сначала форма, потом карта+контакты

### Как определить ТЦ на странице мероприятия

Посмотри как в проекте хранятся данные мероприятия. Скорее всего у каждого мероприятия есть поле `venue` или `tc`. Используй это поле чтобы выбрать нужный объект из `VENUE_CONTACTS` и подставить адрес, телефон и src карты.

Если поля нет — добавь его в структуру данных мероприятий.
